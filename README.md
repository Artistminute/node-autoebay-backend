# Backend Systems

## Getting Started

Clone this repository

```bash
https://github.com/mateothegreat/node-autoebay-backend
```

Now install node_modules

```bash
cd node-autoebay-backend
yarn install
```

Start dev server

```bash
yarn run dev
```
